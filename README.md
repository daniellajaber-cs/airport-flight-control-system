# Airport Flight Control System

Java data structures project that manages flights and landing requests using queues and maps.

## Features
- Add, search, update, and delete flights
- Request landing and manage landing queue
- Emergency landing prioritization
- Display all flights and current landing requests

## Data Structures Used
- HashMap (store flights by flight number)
- LinkedList / Queue (landing requests)

## Technologies
- Java
- Object-Oriented Programming
- Data Structures

## How to Run
1. Open the project in any Java IDE
2. Run the `Main` class (or the menu class)
